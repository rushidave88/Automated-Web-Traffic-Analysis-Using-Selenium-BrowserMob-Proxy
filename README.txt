# SDET Internship Assignment - ExactSpace

## Instructions
1. Install required dependencies:
   pip install selenium browsermob-proxy json

2. Start the script:
   python script.py

3. The script will:
   - Open the ExactSpace website using Selenium.
   - Capture network requests using BrowserMob Proxy.
   - Save network traffic data in `network_data.har`.
   - Parse and analyze HTTP status codes.
   - Generate a summary in `network_summary.json`.

4. Submit the folder as:
   Rushi_Dave_SDET.zip

## Files in Submission:
- script.py (Python automation script)
- network_data.har (Captured network data)
- network_summary.json (Final output)
- Rushi_Dave_Resume.pdf (Your resume)
